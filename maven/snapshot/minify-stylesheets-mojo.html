coffee-mill:minify-stylesheets

Full name:

org.nanoko.coffee-mill:coffee-mill-maven-plugin:1.1.3-SNAPSHOT:minify-stylesheets

Description:

Minifying CSS files.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • Binds by default to the lifecycle phase: package.
