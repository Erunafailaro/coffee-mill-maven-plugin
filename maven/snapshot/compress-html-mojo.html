coffee-mill:compress-html

Full name:

org.nanoko.coffee-mill:coffee-mill-maven-plugin:1.0.1-SNAPSHOT:compress-html

Description:

Compress HTML pages using HtmlCompressor.

Attributes:

  • Requires a Maven project to be executed.

Optional Parameters

Default value is: src/test/coffee.
