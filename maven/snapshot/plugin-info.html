Plugin Documentation

Goals available for this plugin:

Goal Description
coffee-mill:aggregate-javascript Compiles Js files.
coffee-mill:aggregate-stylesheets Compiles less files.
coffee-mill:build-zip The mojo packaging the application.
coffee-mill:compile-coffeescript Compiles coffeescript files.
coffee-mill:compile-css Compiles less files.
coffee-mill:compile-javascript Compiles JavaScript files.
It compiles (checks) JavaScript files from src/main/js.
It copies these JavaScript files into work directory target/www.
This mojo makes the assumption that the files are already copied/generated to their destination directory, when it is executed.
coffee-mill:compile-less Compiles less files.
coffee-mill:compile-sass Compiles less files.
coffee-mill:compress-html Compress HTML files.
coffee-mill:copy-assets Compiles less files.
coffee-mill:documentation-javascript Minifying Js files.
coffee-mill:help Display help information on coffee-mill-maven-plugin.
Call mvn coffee-mill:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
coffee-mill:jasmine-it-test No description.
coffee-mill:jasmine-test No description.
coffee-mill:lint-css Optimize Js files.
coffee-mill:lint-javascript Optimize Js files.
coffee-mill:minify-javascript Minifying Js files.
coffee-mill:minify-stylesheets Minifying Css files.
coffee-mill:optimize-jpeg A processor optimizing JPEG files using JpegTran. JpegTran must be installed on the system and available from the path. -> to install : http://saralinux.blogspot.fr/2013/12/installing-jpegtran-on-mac-or-unixlinux.html -> with brew : brew install jpegtran
coffee-mill:optimize-png A processor optimizing PNG files using OptiPNG. OptiPNG must be installed on the system and available from the path. -> to install : with brew : brew install optipng
coffee-mill:package-assets A processor optimizing JPEG files using JpegTran. JpegTran must be installed on the system and available from the path. -> to install : http://saralinux.blogspot.fr/2013/12/installing-jpegtran-on-mac-or-unixlinux.html
coffee-mill:resolve-dependencies No description.
coffee-mill:test-compile-coffeescript Compiles CoffeeScript files. CoffeeScript files are generally in the src/test/coffee directory. It can be configured using the coffeeScriptTestDir parameter. If the directory does not exist, the compilation is skipped.
coffee-mill:test-compile-javascript Compiles JavaScript Test files.
It compiles (checks) JavaScript files from src/test/js.
It copies these JavaScript files into work directory target/www-test.
This mojo makes the assumption that the files are already copied/generated to their destination directory, when it is executed.
coffee-mill:watch Watch mode Mojo

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven [3.0.3,)
JDK 1.7
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.nanoko.coffee-mill</groupId>
          <artifactId>coffee-mill-maven-plugin</artifactId>
          <version>1.1.0</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.nanoko.coffee-mill</groupId>
        <artifactId>coffee-mill-maven-plugin</artifactId>
        <version>1.1.0</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
