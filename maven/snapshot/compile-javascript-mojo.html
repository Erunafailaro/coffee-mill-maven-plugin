coffee-mill:compile-javascript

Full name:

org.nanoko.coffee-mill:coffee-mill-maven-plugin:1.0.1-SNAPSHOT:compile-javascript

Description:

Copy JavaScript sources to the work directory and check JavaScript sources with
  • Check the code using JSLint
  • Check the code using JSHint
  • Compile dust template (.dust files)

Attributes:

  • Requires a Maven project to be executed.

Optional Parameters

Name Type Since Description
assetsDir File - Where are the assets.
Default value is: src/main/www.
baseDirectory File - Base directory of the project.
buildDirectory File - Directory containing the build files.
coffeeScriptDir File - Where are CoffeeScript files.
Default value is: src/main/coffee.
coffeeScriptTestDir File - Where are CoffeeScript files implementing tests.
Default value is: src/test/coffee.
javaScriptDir File - Where are JavaScript files.
Default value is: src/main/js.
javaScriptTestDir File - Where are JavaScript files implementing tests.
Default value is: src/test/js.
libDir File - Where are the dependencies copies.
Default value is: target/libs.
skipDustCompilation boolean - Enables / disables dust compilation
Default value is: false.
skipJsHint boolean - Enables / disables JSHint
Default value is: false.
skipJsLint boolean - Enables / disables JSLint
Default value is: false.
stylesheetsDir File - Where are LESS, CSS and SASS/SCSS files.
Default value is: src/main/stylesheets.
workDir File - Where are the output files written.
Default value is: target/www.
workTestDir File - Where are the output test files written.
Default value is: target/www-test.
